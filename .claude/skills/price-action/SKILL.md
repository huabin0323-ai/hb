# Al Brooks 价格行为学 — 核心知识图谱

## 思维导图

```
阿布价格行为学 (Price Action)
│
├── 1. 市场结构 (Market Structure)
│   ├── 上升趋势
│   │   ├── 定义: HH + HL 序列（更高高点 + 更高低点）
│   │   ├── 关键: HL 比 HH 更重要（HL 破了趋势就结束了）
│   │   └── 确认: K线收在最高附近、回调1-3根、不碰20EMA
│   │
│   ├── 下降趋势
│   │   ├── 定义: LH + LL 序列（更低高点 + 更低低点）
│   │   ├── 关键: LH 比 LL 更重要
│   │   └── 确认: K线收在最低附近、反弹1-3根、不碰20EMA
│   │
│   ├── 交易区间 (Trading Range)
│   │   ├── 定义: 既不HH+HL也不LH+LL，来回震荡
│   │   ├── 特征: Doji多、K线重叠、20EMA走平
│   │   └── 策略: BLSHS（买低卖高，只做短线）
│   │
│   └── 市场周期 (4阶段循环)
│       ├── 阶段1: 突破 (Spike) — 近乎垂直，几乎无回调
│       ├── 阶段2: 窄通道 (Tight Channel) — 趋势延续，1-3根回调
│       ├── 阶段3: 宽通道 (Broad Channel) — 趋势减弱，5-20根回调
│       └── 阶段4: 交易区间 — 无方向，等待新突破，回到阶段1
│
├── 2. 关键价位与线
│   ├── 20 EMA — 最重要的均线，所有决策都参照它
│   ├── 支撑/阻力 — 前高前低构成的摆动点 (Swing Points)
│   ├── 趋势线 — 连接至少2个摆动低点（牛市）或摆动高点（熊市）
│   └── 通道线 — 趋势线的平行线
│
├── 3. 信号K线 (Signal Bars)
│   ├── 强趋势K线
│   │   ├── 实体 ≥ 总振幅的50%
│   │   ├── 收盘在顶部40%（牛）或底部40%（熊）
│   │   └── 振幅 ≥ 前三根均值的150%
│   │
│   ├── 反转K线 (Pin Bar / 锤子线)
│   │   ├── 创新高/新低
│   │   ├── 长下影/上影（占振幅1/3~1/2）
│   │   └── 收盘在反方向半区内
│   │
│   ├── Inside Bar (内包线)
│   │   ├── ii (两根内包) → 趋势中回调 → 预示延续
│   │   └── iii (三根内包) → 即将突破
│   │
│   └── Outside Bar (外包线)
│       ├── 顺势外包 → 强延续信号
│       └── 逆势外包在极端位 → 潜在反转（但不如Pin Bar可靠）
│
├── 4. 入场信号体系
│   ├── H1/L1 (第一次入场) — 概率40-50%，激进
│   ├── H2/L2 (第二次入场) — 概率~60%，最可靠的顺势入场 ⭐
│   ├── H3/L3 (第三次入场=楔形) — 概率60-70%，反转信号
│   └── H4/L4 — 还没恢复趋势，大概率要反转了
│
├── 5. 形态识别
│   ├── 楔形 (Wedge) — 最重要的形态 ⭐
│   │   ├── 定义: 3次同方向推进 + 2次有意义的回调 + 通道收敛
│   │   ├── 上升楔形 → 看跌反转
│   │   ├── 下降楔形 → 看涨反转
│   │   ├── 趋势中楔形→可能是旗形(延续)而非反转，看上下文！
│   │   └── 楔形失败 → 最强信号之一（突破后1-3根就回来）
│   │
│   ├── 双顶/双底
│   │   ├── 定义非常宽松："接近就够了"，两次尝试即算
│   │   ├── 趋势中: 双底=做多旗形，双顶=做空旗形
│   │   └── 真正反转需3步: 破趋势线 → 测试极值 → 确认跟进
│   │
│   └── 头肩顶/底
│       ├── 本质: 3推形态(=楔形变体)，头=第2推最深
│       └── 大多数头肩形态会失败，失败后是强力顺势信号
│
├── 6. 失败突破 (Failed Breakout) — 80%规则 ⭐
│   ├── 交易区间中: 80%的突破尝试失败 → 反向交易
│   ├── 趋势中: 80%的逆势反转尝试失败 → 顺势交易
│   ├── 突破后收盘回到区间内 = 高概率反向信号
│   └── 机制: 止损猎杀 → 大资金吃掉 → 反向推动
│
├── 7. 测量目标 (Measured Moves)
│   ├── 公式: 目标 = 突破点 + 形态高度
│   ├── 交易区间突破 → 投影区间高度
│   ├── 旗形突破 → 投影旗杆高度
│   └── 腿1=腿2 → 第二腿高度≈第一腿高度
│
├── 8. 风险管理
│   ├── 止损: 信号K线极端外侧1个tick（标准）
│   ├── 交易员公式: (胜率×盈利) > (败率×亏损) 才入场
│   ├── 短线 vs 摆动
│   │   ├── 短线: 盈亏比<1，需70-90%胜率
│   │   └── 摆动: 盈亏比≥2，胜率30-60%即可
│   ├── 仓位: (最大可亏金额) ÷ (每单位止损距离)
│   └── 部分止盈: 在第一个目标位平1/3~1/2，降低压力
│
├── 9. 交易心理
│   ├── "不确定性是市场唯一确定的事"
│   ├── 90%的K线概率只有40-60% → 只在10%的高概率K线交易
│   ├── 感到任何情绪（困惑、兴奋、焦虑、贪婪、恐惧）→ 等待，不交易
│   ├── 永远不要"也许这次会不一样"
│   └── 耐心等待才是真正的优势
│
└── 10. 时间框架
    ├── 5分钟图 — 主要交易图，所有入场/止损/目标在此决定
    ├── 15分钟图 — 趋势确认、20EMA对齐
    ├── 1小时图 — 更高时间框架方向偏见
    ├── 日/周/月图 — 主要支撑阻力、宏观结构
    └── 多时间框架共振 = 最高概率交易
```

## 可编程的核心决策流程

```
1. 判断市场状态:
   IF HH+HL序列 AND 价格>上升20EMA → 牛市趋势
   IF LH+LL序列 AND 价格<下降20EMA → 熊市趋势
   IF 两者都不满足 → 交易区间

2. 判断通道类型（趋势中）:
   IF 回调≤3根 AND 重叠少 → 窄通道（只做顺势）
   IF 回调≥5根 AND 重叠多 → 宽通道（可双向交易）

3. 扫描信号:
   - 信号K线质量（实体比例、收盘位置、振幅）
   - 是第几次入场？（H1/L1概率低，H2/L2是黄金标准）
   - 是否有楔形？（3推+收敛）
   - 是否有失败突破？（80%规则）

4. 计算目标:
   - 测量目标 = 突破点 + 形态高度
   - 检查目标是否在合理范围内

5. 评估风险:
   - 止损位置是否合理？
   - 交易员公式是否为正？
   - 仓位大小是否合适？

6. 执行或放弃:
   IF 全部对齐 → 入场
   ELSE → 等待，市场总会有下一个机会
```

## 关键概率速查表

| 场景 | 概率 | 策略 |
|------|------|------|
| 交易区间突破 | 80%失败 | 反向交易突破 |
| 趋势中逆势反转 | 80%失败 | 顺势交易 |
| H2/L2 顺势二次入场 | ~60%成功 | 最可靠入场点 |
| 楔形H3/L3反转 | 60-70%成功 | 高概率反转 |
| 窄通道顺势 | 高 | 只顺势，不逆势 |
| 窄通道最终下破 | 75%概率 | 突破后会变宽或变区间 |
| 宽通道回归起点 | 75%概率 | 目标设回通道起点 |
| 突破模式(BOM)首次突破 | ~50% | 比普通区间突破更可靠 |
