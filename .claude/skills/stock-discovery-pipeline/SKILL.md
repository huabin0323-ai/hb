# PA Stock Discovery Pipeline — A股价格行为学每日全量扫描+深度评审

## PA策略来源

**核心框架：** Al Brooks 价格行为学（`price-action` skill + `hb-trading/src/price_action.py`）

**策略增强：** 从小红书/公众号持续抓取的A股PA实战策略（`xhs-scraper/` → `trading-learn` skill → 策略融合），后续迭代接入。

**当前状态：** Phase 0-5 Python代码已完成，小红书策略接入后自动更新信号权重和过滤规则。

## 每日产出清单

| # | 产出 | 内容 | 推送飞书 |
|---|------|------|:---:|
| 1 | **大盘行情概览** | 主要指数、板块轮动、市场情绪、宏观背景 | ✅ |
| 2 | **200只PA全量决策矩阵** | 每只买入/不动决策 + 次日胜率/混淆矩阵 | ✅ |
| 3 | **TOP 5 五角色深度评审** | 5角色PA委员会并行分析每只TOP5 | ✅ |
| 4 | **TOP 5 正式报告** | 含PA理由、入场/止损/止盈、证伪条件 | ✅ 文档链接 |
| 5 | **昨日回顾** | 全量混淆矩阵 + 信号类型胜率拆解 + 修正建议 | ✅ |

## Phase 4: TOP 5 五角色PA委员会深度评审 + 正式报告

### 4.1 五角色PA委员会

对 TOP 5 中的**每一只**股票，启动5个独立角色并行分析，每个角色从自己的专业角度给出判断：

```
角色1: 市场结构师 (Market Structure Analyst)
角色2: 信号K线分析师 (Signal Bar Analyst)  
角色3: 量能资金分析师 (Volume & Flow Analyst)
角色4: 风险控制官 (Risk Officer)
角色5: 首席交易员 (Head Trader) → 最终综合
```

#### 角色1: 市场结构师

```
职责: 判定日线/周线市场结构，确认趋势方向和阶段

分析维度:
  - 摆动结构: HH+HL / LH+LL 序列分析
  - 趋势阶段: 阶段1/2/3/4 判定及依据
  - 通道类型: 窄/宽，回调特征
  - 关键价位: 支撑/阻力/20EMA/50EMA/趋势线
  - 多TF验证: 60min/日线/周线方向一致性

输出:
  - 市场结构评分: 0-100
  - 结构描述: (自然语言，说明为什么是这样的结构)
  - 结构风险: 什么情况下结构会被破坏？
```

#### 角色2: 信号K线分析师

```
职责: 识别和评估PA信号的质量和可靠性

分析维度:
  - 信号类型: H2/楔形/失败突破/平台突破/Pin Bar...
  - 信号质量: 实体比例、收盘位置、振幅、与前K线关系
  - 入场序列: 是第几次入场？H1/L1还是H2/L2？
  - 信号位置: 信号出现在结构的什么位置？有利还是不利？
  - 历史验证: 同类型信号在该股过去的成功率？

输出:
  - 信号质量评分: 0-100
  - 信号描述: (自然语言，详细说明识别依据)
  - 信号失效条件: 什么K线形态出现则信号作废？
```

#### 角色3: 量能资金分析师

```
职责: 分析成交量和资金流向，确认信号是否有资金支撑

分析维度:
  - 量价配合: 放量上涨/缩量回调 是否符合PA逻辑
  - 量能结构: 5日/20日均量，量比，量堆
  - 资金流向: 主力净流入/流出，大单动向
  - 换手率: 是否合理（过低=无人气，过高=对倒风险）
  - 筹码分布: 当前位置的套牢盘/获利盘比例

输出:
  - 量能评分: 0-100
  - 量能描述: (自然语言)
  - 量能风险: 什么量能变化预示信号失效？
```

#### 角色4: 风险控制官

```
职责: 全面评估该交易计划的风险，决定是否值得执行

分析维度:
  - R:R计算: 测量目标 vs 止损距离，盈亏比是否达标
  - 止损合理性: 止损位是否在PA结构关键位外侧
  - A股特有风险: 减持/解禁/业绩/庄股/涨停板风险
  - 仓位建议: 基于2%单笔风险上限的具体手数
  - 黑天鹅评估: 如果明天大盘跌2%，这笔交易会怎样？

输出:
  - 风险评分: 0-100 (100=风险最低)
  - 风险描述: (自然语言，列出所有风险点)
  - 风控建议: 具体止损/仓位/应急方案
```

#### 角色5: 首席交易员（综合评审）

```
职责: 综合前4个角色的分析，给出最终决策和交易计划

综合维度:
  - 各角色评分加权: 结构30% + 信号30% + 量能20% + 风控20%
  - 矛盾处理: 如果角色间意见冲突，说明冲突点和处理逻辑
  - 信念度: 基于多角色一致性的确信度
  - 交易计划: 具体的入场/止损/止盈价位
  - PA策略符合性: 为什么这笔交易符合Al Brooks方法论

输出:
  - 综合评分: 0-100
  - 最终决策: 强烈买入 / 买入 / 谨慎买入
  - PA符合理由: (这是最关键的部分)
      1. 结构层面 — 为什么市场结构支持这个交易
      2. 信号层面 — 具体的PA信号是什么，为什么可靠
      3. 量能层面 — 资金是否确认信号
      4. 风险层面 — 为什么R:R值得参与
  - 交易计划: 入场价/止损价/止盈1/止盈2/仓位
  - 证伪条件: 什么情况下判断错误，立即退出
```

## 核心约束

1. **不构成投资建议** — 所有输出醒目标注
2. **全量决策** — 200只每只必须有决策，不选TOP N自欺欺人
3. **五角色独立** — 5个角色并行分析，不允许串通，冲突必须显式记录
4. **PA理由完整** — 每只TOP5必须从结构/信号/量能/风险四个层面说明PA依据
5. **数据驱动修正** — 策略参数调整基于混淆矩阵数据
6. **可证伪** — 每个买入计划附带证伪条件
7. **飞书必达** — 每日3次推送，失败重试1次，仍失败记录日志
