# 交易知识学习引擎

## 触发条件

用户发送以下内容时自动启用：
- 小红书帖子链接 (xiaohongshu.com)
- 公众号文章链接
- 交易类书籍/文章文本
- K线图截图（带标注的图表分析）
- 知乎/WeChat/任何平台的交易分析内容

## 处理流程

1. **接收内容** — 用户发链接、文字或图片
2. **内容提取** — WebFetch 尝试获取（链接）；Read 工具读取图片（截图/图表）
3. **核心观点提取** — 找到：交易逻辑、入场理由、止损位、目标位、市场背景
4. **策略映射** — 与现有框架对比：
   - 阿布价格行为学：是 HH/HL 序列？楔形？二次入场？80% 规则？
   - 情绪面：市场处于恐慌还是贪婪？
   - 宏观面：是否有政策/事件配合？
5. **存入案例库** — 保存到 `D:/hb/hb-trading/data/cases/`，编号归档
6. **评分** — 给这个案例的可学性打分（1-5）

## 输出格式

```
## 案例 #{id}: {标题}
来源: {url}
日期: {date}

### 原文核心观点
{提取的 3-5 条核心观点}

### 价格行为学分析
- 市场结构: {趋势/区间/通道}
- 信号类型: {H2/L2/楔形/失败突破/...}
- 入场位置: {具体价位和逻辑}
- 止损/目标: {原文给出的或推算的}

### 与我们的策略对照
- 符合/不符合 80% 规则？
- 是第几次入场？
- 情绪面是否配合？
- 我们会在同样的位置入场吗？

### 可学评分: ⭐⭐⭐⭐⭐
{一句话总结学到了什么}
```
