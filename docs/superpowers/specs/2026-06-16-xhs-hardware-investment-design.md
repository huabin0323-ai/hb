# 小红书起号：硬科技投资笔记 — 设计文档

> 2026-06-16 | 版本 v1.0

## 一、项目概述

**账号定位：** 一个华为硬件工程师的硬科技投资笔记。以个人学习为主线，从产业链和投资视角分析硬件技术，提供技术→投资的翻译层。

**两个核心支柱：**
- **学硬件，公开学** — 高速接口、协议、芯片架构、仿真测试，学到什么写什么
- **产业链投资分析** — 技术变化→供应链谁受益→A股谁在做

**不做什么：**
- 不荐股、不喊单（合规边界）
- 不建群、不卖课、不引流私域
- 不招聘（不主动提及华为职位和团队信息）
- 不追纯热点（没有产业逻辑支撑的热点不碰）

**差异化护城河：**
小红书讲半导体的99%是财经博主，他们说不清楚112G SerDes对PCB材料损耗要求的区别。你能从信号完整性仿真推导出谁的高频覆铜板更可能拿到订单——这个壁垒财经博主跨不过来，硬件工程师不爱写。

## 三、内容体系

### 3.1 四个内容类型

| 类型 | 触发条件 | 投资视角怎么融入 | 频率 |
|---|---|---|---|
| **学概念** | 今天搞懂一个技术点 | 这个概念卡了谁脖子/谁在做 | 1篇/周 |
| **拆产业链** | 系统性研究一个产业链 | 每环标A股标的+竞争格局+毛利率 | 1篇/周 |
| **跟事件** | 涨价/缺货/新技术/业绩 | 用产业链逻辑判断谁真受益 | 1篇/2周 |
| **做对比** | 三条路径/三家公司需要选 | 技术路线差异→投资逻辑差异 | 1篇/2周 |

### 3.2 内容公式（每篇结构）

```
1. 一个问题/概念（读者一看就懂的痛点）
2. 技术解读（大白话 + 图解，为什么是这样）
3. 产业链映射（这个变化涉及哪些环节，谁受益谁受损）
4. A股相关标的 + 差异分析（"谁在做什么"，不做"买什么"）
5. 一句话总结 + 建议观察点
```

### 3.3 视觉风格

- 归藏/Swiss Style（guizang-social-card-skill 生成）
- 产业链全景图为竖版3:4（适配小红书图文）
- 技术对比为表格/决策卡形式
- 每篇1-6张图（封面1张，内容图1-5张）

### 3.4 前10篇选题

**学概念系列（3篇）：**
1. 搞懂S参数 — 为什么做高速设计必须会看Smith图 → SI仿真工具链谁能国产替代
2. 搞懂眼图 — 一根信号线能跑多快，物理层说了算 → 高速测试仪器/EDA的A股图谱
3. 搞懂PCIe Retimer — 信号在路上"累"了怎么办 → Retimer芯片竞争格局

**拆产业链系列（3篇）：**
4. HBM产业链全景：从TSV→堆叠→CoWoS→A股谁在摸哪个环节
5. 光模块产业链全景：光芯片/DSP/封装，每层国产化程度完全不同
6. 先进封装产业链全景：CoWoS/EMIB/混合键合，Foundry和OSAT的边界在消失

**跟事件系列（2篇）：**
7. 铜连接概念火了 — GB300到底用了什么，A股谁是真链谁是蹭概念
8. 某存储/AI芯片公司财报 — 从产业链位置看业绩能否持续

**做对比系列（2篇）：**
9. CPO vs LPO vs 可插拔 — 光模块三条路线，对应投资逻辑完全不一样
10. AI服务器PCB三家：深南/沪电/景旺 — 层数涨了谁吃肉

## 四、内容生产管线

### 4.1 输入层

| 来源 | 具体渠道 | 用途 |
|---|---|---|
| 个人学习 | 仿真笔记、协议阅读、课程笔记 | 学概念系列素材 |
| 券商研报 | 半导体/电子行业研报 | 产业链数据、市场规模 |
| 招股书/年报 | 科创板半导体公司 | 竞争格局、客户结构 |
| 行业新闻 | 电子工程专辑/集微网/半导体行业观察 | 事件驱动素材 |
| A股数据 | 行情软件/公告 | 标的筛选、财务对比 |

### 4.2 加工层

```
输入素材 → 技术翻译（工程语言→投资语言）
         → 产业链映射（技术变化→谁受益/受损）
         → 标的对比（技术路线差异→竞争格局差异）
         → 图文产出（guizang-social-card-skill）
```

### 4.3 输出

- 小红书图文（竖版3:4，正文800-1200字）
- 产业链全景图（信息图，单张或多张轮播）
- 技术对比表/决策卡

### 4.4 发布节奏

- 前两周：日更（7篇），快速建立标签权重
- 稳定期：3-4篇/周，四种类型轮转

## 五、起号阶段规划

| 阶段 | 粉丝量 | 核心目标 | 内容侧重 | 关键动作 |
|---|---|---|---|---|
| 冷启动 | 0-500 | 打标签，拿搜索流量 | 产业链拆解 + 事件解读 | 标题优化，关键词覆盖 |
| 积累期 | 500-2000 | 展示深度，建立信任 | 学概念 + 做对比 | 评论区互动，回复专业问题 |
| 稳定期 | 2000+ | 持续输出，成为参考源 | 四种类型均衡 | 尝试跨平台分发 |

**不做数据焦虑的事：** 不追热点蹭流量，不标题党，不用互粉互赞。冷启动慢一点没关系，优先保证内容质量和技术判断的准确性。

## 六、合规边界

- 所有涉及A股标的的内容，表述为"梳理产业链公司"而非"推荐买入"
- 不预测股价、不给目标价、不说"肯定涨"
- 涉及华为工作时，只说公开信息（行业趋势判断），不涉及任何内部数据、未公开产品、客户信息
- 引用数据注明来源（研报/招股书/公告）

---

## 七、成功指标

- **质量指标：** 收藏率 > 点赞率（产业链图解类内容的正常模式）
- **信任指标：** 评论区出现专业讨论，而非纯"求带""怎么买"
- **长期指标：** 行业人士关注、引用、转发
- **不发监测的指标：** 粉丝数增长速度（冷启动期不做KPI）
