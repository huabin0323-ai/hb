# /xhs-scrape

爬取指定链接的内容（自动识别小红书/微信公众号），提取正文+全部多级评论，并按价格行为学策略框架提炼精华。

## 用法

```
/xhs-scrape <url>
```

## 执行流程

1. 运行 `python cli.py "<url>"` 启动爬虫
2. 爬虫自动：
   - 识别平台（小红书/公众号）
   - 启动已登录的 Edge 浏览器
   - 提取正文 + 图片
   - 小红书：拦截评论 API，递归抓取全部多级嵌套评论
   - 公众号：只需提取文章正文
   - 保存到 `output/{date}_{title}/content.md`
3. CLI 输出爬取结果摘要（标题、字数、评论数）
4. **你（Claude）读取 `output/{dir}/content.md`** 的内容
5. **按以下模板提炼精华**，写入 `output/{dir}/summary.md`，并展示给用户

## AI 总结模板

```markdown
# [标题] 策略精华
> 来源: 小红书/公众号 | 作者: xxx | 日期: xxx

## 涉及的价格行为理论
- 识别文中提到/隐含的 Al Brooks 理论编号
  （如 #19支撑背叛、#23熊棒做多、H2/L2入场、80%规则等）

## 策略观点提炼
1. **核心论点** — 作者的主张
2. **逻辑链** — 前提→推理→结论
3. **适用场景** — 趋势/震荡/突破？什么品种/时间框架？

## 入场/出场逻辑
- 信号触发条件
- 止损/止盈设置思路
- 仓位管理观点

## 评论碰撞（仅小红书）
### 支持/印证
- 评论中其他人的实盘验证
### 质疑/反例
- 不同意见及其论据
### 补充细节
- 评论中补充的关键参数、边界条件

## 与现有策略框架的关联
- 对 hb-trading 系统的参考价值
- 可吸收的参数/规则

## 标签
#价格行为学 #AlBrooks #交易策略 #...
```

## 示例

```
/xhs-scrape https://www.xiaohongshu.com/explore/abc123def456
/xhs-scrape https://mp.weixin.qq.com/s/xxxx
```
