## 二、逐策略详细设计

### 策略1: EMA20 动态支撑/阻力

**理论**: 趋势中 EMA20 是动态支撑/阻力。价格测试 EMA20 失败→趋势延续；突破 EMA20→趋势可能反转。

**案例26原文**:
> "在K5和K6处，两次试图突破EMA20均线，都没有获得成功...用阿布失败再失败理论，可以预期，行情将转头向下"

**实现** (`price_action.py` 新增函数):
```python
def detect_ema_interaction(df: pd.DataFrame, state: MarketState) -> list[EntrySignal]:
    """检测价格与EMA20的互动信号
    - 趋势中回踩EMA20反弹 → 趋势延续入场
    - 两次测试EMA20失败 → 反转信号
    """
    # 计算EMA20
    # 检测最近K线是否触及EMA20
    # 上涨趋势中：价格回踩EMA20反弹 → long entry
    # 上涨趋势中：两次测试EMA20都失败 → 反转short预警
```

### 策略2: "失败再失败" 反转

**理论**: 某个关键位置第一次突破失败→正常；第二次再失败→强反转信号。

**案例26原文**:
> "两次试图突破EMA20均线，都没有获得成功，用阿布失败再失败理论，可以预期，行情将转头向下"

**实现** — 可合并到 `detect_failed_breakouts()`:
- 当前只检测单次假突破
- 新增：检测同一阻力位的两次突破失败 → 置信度x2

### 策略3: 双底牛旗

**理论**: 两低点相近、中间有更高低点 → 双底支撑。如案例22: K3和K4构成双底。

**案例22原文**:
> "K3和K4还构成了双底牛旗"

**实现** (`price_action.py` 新增):
```python
def detect_double_bottom(df: pd.DataFrame, swings: list[SwingPoint]) -> list[EntrySignal]:

    - 两个 swing low 价格接近（差距<ATR*0.3）
    - 中间有更高的 swing low
    - 双底之间至少隔5根K线
    """
```

### 策略4: 通道边界交易

**理论**: 宽/窄通道中，价格在上下沿之间摆动。下沿做多，上沿止盈。

**案例26原文**:
> "K2跌至通道下沿附近，可以预期在此处获得支撑"

**实现**: `analyze_structure()` 已计算 channel_top/bottom，但未用于信号。新增：

```python
    """价格触及通道边界 → 入场/出场信号"""
```

### 策略5: 孕线=衰竭

**案例22原文**:
> "K5是K4的孕线，标志着空头的乏力"

**实现**: `detect_signal_bars()` 已检测 inside_bar，但未赋予"衰竭"语义。增强：inside_bar 出现在趋势尾部→标记为衰竭信号。

### 策略6: 水平压力/支撑位

**案例22原文**:
> "到K7的上涨，突破了由多根上影线构成的水平压力位，我预计突破会失败"

**实现**: 检测多根K线上影线在同一价格区间的聚集 → 形成水平压力位。

### 策略7: 趋势恢复日模型

**案例21原文**:
> "此票还是一个'趋势恢复交易日'模型...当天进场，收盘获利4个点"

**特征**: 强势上涨→小幅回调→某天开盘后迅速突破昨日高点→趋势恢复。需要日内结构判断，较复杂。

### 策略8: 楔形分类（反转 vs 持续）

**案例21原文**:
> "强势上涨之前，是一个三推楔形牛旗（反转形态牛旗）...强势上涨之后，是一个弱势回调至均线的三推楔形牛旗（持续形态牛旗）"

**实现**: `detect_wedges()` 增加分类字段 — 楔形在大趋势中的位置决定性质。
